# pj17_combo_sentiment_embedding_trade

Комбинирование стратегий торговли фьючерсами на московской бирже.

## Описание

Этот проект объединяет две торговые стратегии для автоматической торговли фьючерсами на MOEX:

- **Эмбеддинги новостей**: Анализ RSS новостей о нефти и газу с использованием векторных эмбеддингов (Ollama) для предсказания направления цены.
- **Сентимент-анализ**: Оценка настроения новостей через локальную LLM для генерации торговых сигналов.

Проект включает скачивание данных, анализ, бэктест, валидацию и генерацию сигналов для QUIK.

## Структура

- `rts/`, `mix/` - Папки по инструментам
- `beget/` - Сбор новостей с сервера
- `run_all.py` - Запуск всего пайплайна
- `requirements.txt` - Зависимости

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте Ollama и QUIK

## Запуск

```bash
python run_all.py
```
